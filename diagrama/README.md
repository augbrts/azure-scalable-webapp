# Diagrama de arquitetura

- **`arquitetura.drawio`** — diagrama editável (abra em https://app.diagrams.net).
- **`arquitetura.png`** — exporte o PNG a partir do `.drawio` (File → Export as → PNG) e
  salve aqui com este nome (o README principal referencia `diagrama/arquitetura.png`).

O diagrama inicial já contém os elementos exigidos pelo enunciado: usuário/Internet, IP público,
Load Balancer, VM Scale Set, sub-redes pública e privada, banco gerenciado, object storage,
as regras de segurança (NSGs) e a região. Ajuste o visual como preferir antes de exportar.
